How it works:

The expandCollapse animation uses state transitions to:

  • Define visual properties for both 'collapsed' and 'expanded' states
  • Animate height, opacity, and padding simultaneously
  • Use bidirectional transitions between states with ease-in-out timing
  • Apply a transform counter-balance (translateX) that offsets the padding expansion, keeping text visually centered during the animation
  • Create a more refined effect where the container expands around the content rather than pushing the content
