How it works:

In reusable-animations.ts, we define animations with animation() function that can accept parameters:

  • Create reusable animations using animation() function
  • Use useAnimation() to apply these animations in components
  • Override default animation parameters when applying
  • Define different timing functions, durations and effects

This approach lets you maintain animations in a central place for consistency.
