How it works:

In reusable-animations.ts, we define animations with animation() function that can accept parameters:

  • Create reusable fadeIn/fadeOut animations using animation() function
  • Use useAnimation() to apply these animations with different parameters
  • Override default durations (from 300ms to 2000ms)
  • Apply different easing functions (ease-in, ease-out, cubic-bezier)
  • Using trackBy to improve animation performance during restoration

This approach lets you maintain animations in a central place for consistency while customizing them for each use case.
