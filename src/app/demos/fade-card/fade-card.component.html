Animation Challenge:

Complete the following tasks to implement the card animations:

  • Create a fadeAnimation trigger in the component
  • Implement :enter transition for new cards
  • Implement :leave transition for removed cards
  • Apply the animation trigger to each card in the template
  • Use [@.disabled] to prevent animations on initial render
  • Test by adding and removing cards

The goal is to create smooth animations when adding and removing cards, while preventing unwanted animations when the page first loads.
